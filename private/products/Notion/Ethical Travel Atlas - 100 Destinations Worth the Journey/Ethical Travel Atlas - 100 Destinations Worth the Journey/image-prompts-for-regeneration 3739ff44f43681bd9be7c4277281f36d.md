# Image Generation Prompts (for upgrading the 20 illustrated SVG maps)

Ten of the 30 destinations ship with full hand-painted photorealistic isometric maps generated by an image model. The other twenty ship with on-brand illustrated SVG cards (converted to PNG) that are sales-ready as-is.

If you want to upgrade all 30 to the same photorealistic style later, paste the prompts below into your image-generation tool of choice (any modern model like GPT-4o image, Midjourney, Flux, etc. works). Save each as `images/<slug>.png` to overwrite the current placeholder. The Markdown pages will pick the new image up automatically because they reference `../images/<slug>.png`.

The 10 already-generated photorealistic maps are: Kyoto, Luang Prabang, Chiang Mai, Hoi An, Pokhara, Kerala Backwaters, Paro Valley, Tbilisi, Istanbul, Petra.
